# Tetris

A simple Tetris game made with JavaFX. 


:sparkles: Features


# Installation

## Prerequisities
- Installation of the JDK with version 21 or higher

>[!NOTE]
make sure that java is installed by typing inside your terminal `java --version`

1. Clone the repository
    ```zsh
   https://github.com/Armin-Keric/Tetris.git
   ```
2. Navigate into the project directory

```
cd Tetris
```

3. Run the program

```zsh
# On Windows (Command Prompt/PowerShell)
.\mvnw javafx:run
```

```zsh
# On Linux/macOS
chmod +x mvnw
./mvnw javafx:run
```
